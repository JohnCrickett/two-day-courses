Exercise 3: Pointers and Arrays

1. Working with structures

Write a date structure, eg something like: 
struct date
{
    int year;
    int month;
    int day;
};
Make a function date_print() that can print such a date object. Create three date objects, one with global static storage class, one with local automatic storage class, and one allocated on the free store (malloc). Practice using designators and struct compound literals.

Write a date_equal() function that takes two date objects and returns true if they are equal.

Try different ways of passing date objects to date_print() and date_equal().

2. Write a function that adds two vectors

Write a function addvec() that can be used like this: 
double a[4] = {4.2,5.9,3.2,4.5};
double b[4] = {8.9,-2.1,0.3,5.6};
double c[4];
addvec(4, c, b, a); // c = b+a;
Try to measure how fast this function is for adding large vectors. Try different things to optimize your code. Does using the restrict type qualifier have any effect?
