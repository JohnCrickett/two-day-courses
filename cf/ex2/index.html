Exercise 2: Types, Operators, Expressions and Control Flow

1. Counting Bits

Write a program that counts the number of set bits in a given integer. Start by writing some unit tests. 
$ bitcount 1
1
$ bitcount 14
3
$ bitcount 256
1
if no argument is given print a diagnostic message as follows: 
$ bitcount
usage: bitcount value

2. Calculate statistics over some numbers

Write a program that calculate some simple statistics for a sequence of integers. Eg, 
$ cat foo.dat
4
9
3
$ cat foo.dat | calcstat 
n=3, average=5.333333, min=3, max=9
Write unit tests when developing the code.

Extra: Extend the calcstat program to accept an argument, a file name, so that you can do: 

$ calcstat foo.dat
n=3, average=5.333333, min=3, max=9
