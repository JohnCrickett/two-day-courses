Exercise 1: A Brief Tour

1. Hello World!

Compile and run a program that prints out "Hello World!".

2. Print Arguments

Write a program that prints out all the arguments given at the command line.

3. Linking together a program

Declare a function 'void sayhello(void);' in a header file, sayhello.h, then implement it in sayhello.c. Compile sayhello.c into sayhello.o

Then write a main.c which includes sayhello.h and calls the function. Compile main.c into main.o.

Link sayhello.o and main.o together to get an executable that you can run.

Eg, 

  gcc -c sayhello.c
  gcc -c main.c
  gcc -o main sayhello.o main.o
