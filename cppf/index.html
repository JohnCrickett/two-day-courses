C++ Foundation - a 2 day course

This is a 2 day course in C++.

Prerequisite

This course assumes that you are already familiar with C++, and that you have done the C Foundation course.

Preparations

Please make sure that you all have a computer that can compile "Hello C++!" without the need of a network connection. Also, make sure that have a decent code editor installed and knows how to use it properly.

Also, it would be nice if those of you who do not program C++ regularly prepare yourself a bit extra by writing some snippets of C++ where you experiment with things like constructors, destructors, using the vector class, namespaces, exceptions, and so on. Oh, by the way, please bring a copy Bjarne Stroustrups The C++ Programming Language.
