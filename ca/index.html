C Advanced - a 2 day course

This is a follow up course for those who have already done the 2 day C Foundation course. This time the course will focus more on the advanced features of the C programming language, things that you need to know to write solid and usable modules in C. The course will be based on the current C Standard (C99).

Prerequisite

The course assumes that you have done the C Foundation course.

Plese bring a computer with a C99 compliant compiler and a decent editor. Please also bring a USB stick if you have one.
